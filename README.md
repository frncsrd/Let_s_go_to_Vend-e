# Checklist Vente / Achat 2026

Application web simple en HTML/CSS/JS pur.

## Fonctionnalités

- Checklist par période
- Ajout de tâches personnalisées
- Sauvegarde automatique (localStorage)
- Barre de progression
- Notes libres

## Utilisation

1. Ouvrir `index.html` dans un navigateur
2. Cocher les tâches
3. Ajouter des éléments si nécessaire

## Déploiement GitHub Pages

1. Push sur GitHub
2. Settings → Pages
3. Source : Deploy from branch → main
4. Accès via l’URL GitHub Pages
